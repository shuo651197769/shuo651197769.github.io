Assuming Sublime Text has been installed in the Applications folder, and then you should make a link to *subl* you can run:

{% highlight java %}
ln -s "/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl" /usr/bin/subl
{% endhighlight %}
